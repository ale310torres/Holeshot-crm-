HOLESHOT CRM - SUBIDA A GITHUB

Esta carpeta esta lista para subir a GitHub.

Sube todo el contenido de la carpeta, no la carpeta como una carpeta anidada.

No subas node_modules.
No necesitas subir dist para Vercel; Vercel lo construye solo.

Despues de subir:
1. En Supabase, ejecuta supabase/schema.sql completo.
2. En Supabase Authentication, crea el usuario que entrara al CRM.
3. En Vercel, conecta este repo.
4. Agrega las variables:
   VITE_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY
   VITE_APP_NAME=Holeshot CRM
5. Deploy.

Esta version esta alineada a Holeshot: servicio, piezas, motoras, ATV,
cotizaciones, tareas, pagos, ordenes de servicio y seguimiento.
